Sage - ML Analysis Core

A comprehensive machine learning analysis toolkit for sentiment analysis, data processing, and model training.

Features

  • Sentiment Analysis: Multiple backends (TextBlob, NLTK, BERT) with confidence scoring
  • Data Analysis: Comprehensive data analysis and reporting tools
  • Model Training: Utilities for training and managing ML models
  • Data Validation: Quality assessment and validation tools
  • Report Generation: Markdown and JSON report generation

Installation

pip install sage

Quick Start

Sentiment Analysis

from sage import SentimentAnalyzer, EnhancedSentimentAnalyzer

# Basic sentiment analysis
analyzer = SentimentAnalyzer()
result = analyzer.analyze_text("I love this product! It's amazing.")

print(result)
# Output: {
#     'sentiment': 'positive',
#     'scores': {'positive': 0.8, 'negative': 0.0, 'neutral': 0.2, 'compound': 0.8},
#     'confidence': 1.0,
#     'methods_used': ['textblob', 'nltk']
# }

# Enhanced analysis with additional features
enhanced_analyzer = EnhancedSentimentAnalyzer()
enhanced_result = enhanced_analyzer.analyze_text("This is a great product!")

print(enhanced_result)
# Output includes additional features like text statistics and confidence scoring

Components

Sentiment Analysis

  • SentimentAnalyzer: Main sentiment analyzer with multiple backends
  • TextBlobAnalyzer: TextBlob-based sentiment analysis
  • NLTKAnalyzer: NLTK VADER sentiment analysis
  • BertAnalyzer: BERT-based sentiment analysis (placeholder)
  • EnhancedSentimentAnalyzer: Advanced features and confidence scoring

Analysis & Reporting

  • AnalysisReporter: Generate reports from sentiment analysis results
  • MarkdownGenerator: Create comprehensive markdown reports
  • SentimentStatistics: Calculate statistics from sentiment data
  • DataAnalyzer: Analyze various data types
  • StructuralAnalyzer: Analyze data structure and patterns
  • TrendAnalyzer: Analyze trends in data over time
  • SimilarityNetwork: Analyze similarity between data points
  • TopicModeling: Discover topics in text data
  • Summarizer: Generate text summaries

Training

  • TrainingDataManager: Organize and manage training datasets
  • ModelTrainer: Train machine learning models
  • ModelEvaluator: Evaluate trained models
  • ModelManager: Manage trained models

Utilities

  • SageConfig: Configuration management with Pydantic
  • DataValidator: Validate data for processing
  • QualityValidator: Assess data quality
  • Logging: Comprehensive logging utilities

Configuration

Sage uses Pydantic for configuration management. You can configure it through environment variables or programmatically:

from sage import SageConfig

# Use default configuration
config = SageConfig()

# Customize configuration
config = SageConfig(
    sentiment_min_text_length=20,
    training_batch_size=64,
    log_level="DEBUG"
)

# Environment variables (prefixed with SAGE_)
# SAGE_SENTIMENT_MIN_TEXT_LENGTH=20
# SAGE_TRAINING_BATCH_SIZE=64
# SAGE_LOG_LEVEL=DEBUG

Examples

Batch Sentiment Analysis

from sage import SentimentAnalyzer

analyzer = SentimentAnalyzer()

# Analyze multiple comments
comments = [
    {"text": "Great product!", "author": "user1"},
    {"text": "Not satisfied with the quality.", "author": "user2"},
    {"text": "Amazing service and fast delivery.", "author": "user3"}
]

analyzed_comments = analyzer.analyze_comments(comments)

# Save results
analyzer.save_results(analyzed_comments, "sentiment_results.json")

Comprehensive Analysis

from sage import (
    SentimentAnalyzer, 
    AnalysisReporter, 
    MarkdownGenerator,
    SentimentStatistics
)

# Perform sentiment analysis
analyzer = SentimentAnalyzer()
articles_with_sentiment = analyzer.analyze_articles(articles)

# Generate statistics
stats = SentimentStatistics()
overall_stats = stats.calculate_overall_statistics(articles_with_sentiment)

# Create reports
reporter = AnalysisReporter()
reporter.generate_markdown_report(
    articles_with_sentiment,
    "comprehensive_report.md",
    "Comprehensive Sentiment Analysis Report"
)

# Generate markdown summary
generator = MarkdownGenerator()
generator.generate_articles_summary(
    articles,
    "articles_summary.md"
)

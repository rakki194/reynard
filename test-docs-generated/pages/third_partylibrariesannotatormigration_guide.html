Annotator Library Migration Guide

This guide explains how to migrate from the existing caption generation systems in yipyap and pawprint to use the unified annotator library.

Overview

The annotator library provides a unified API for image captioning and tagging, consolidating functionality from:

  • yipyap/app/caption_generation/
  • yipyap/app/services/caption_generator.py
  • utils/caption/ (already extracted)

Migration Steps

1. Install the Annotator Library

First, install the annotator library in your project:

# From the libraries/annotator directory
pip install -e .

# Or add to requirements.txt
annotator @ file:///path/to/libraries/annotator

2. Update Yipyap Integration

Replace Caption Generator Service

Replace yipyap/app/services/caption_generator.py with:

"""
Caption Generator Service using Annotator Library

This module provides the CaptionGeneratorService that manages the Annotator
library and provides caption generation functionality through the service management system.
"""

import logging
from typing import Optional, Dict, Any, List

from .base import BaseService
from .health import ServiceHealth

# Import from annotator library
from annotator import AnnotatorService

logger = logging.getLogger("uvicorn")


class CaptionGeneratorService(BaseService):
    """
    Service for managing the Annotator library.

    This service handles the initialization and lifecycle management of the
    caption generation system using the unified annotator library.
    """

    def __init__(self):
        """Initialize the caption generator service."""
        super().__init__(
            name="caption_generator",
            dependencies=["image_processing"],  # Depends on image processing service
            required_packages=["annotator"],  # Require annotator library
            startup_priority=6,  # After model registry and image processing
            health_check_interval=60,  # Check every minute
        )
        self._annotator_service = None
        self._available_captioners: Dict[str, Dict[str, Any]] = {}

    async def initialize(self) -> bool:
        """Initialize the caption generator service."""
        try:
            logger.info("Initializing caption generator service with annotator library")

            # Initialize annotator service
            self._annotator_service = AnnotatorService()
            success = await self._annotator_service.initialize()

            if success:
                # Get available captioners
                self._available_captioners = (
                    self._annotator_service.get_available_captioners()
                )

                logger.info(
                    f"Caption generator service initialized with {len(self._available_captioners)} available captioners"
                )
                return True
            else:
                logger.error("Failed to initialize annotator service")
                return False

        except Exception as e:
            logger.error(f"Failed to initialize caption generator service: {e}")
            return False

    async def shutdown(self) -> None:
        """Shutdown the caption generator service."""
        try:
            logger.info("Shutting down caption generator service")

            if self._annotator_service:
                await self._annotator_service.shutdown()
                self._annotator_service = None

            self._available_captioners.clear()

        except Exception as e:
            logger.error(f"Error shutting down caption generator service: {e}")

    async def _health_check(self) -> ServiceHealth:
        """Perform health check on the caption generator service."""
        try:
            if not self._annotator_service:
                return ServiceHealth.UNHEALTHY

            # Check if any captioners are available
            if not self._available_captioners:
                return ServiceHealth.DEGRADED

            # Check if at least one captioner is ready
            ready_captioners = [
                name
                for name, info in self._available_captioners.items()
                if info.get("available", False)
            ]

            if not ready_captioners:
                return ServiceHealth.DEGRADED

            return ServiceHealth.HEALTHY

        except Exception as e:
            logger.error(f"Caption generator health check failed: {e}")
            return ServiceHealth.UNHEALTHY

    def get_captioner_manager(self):
        """Get the captioner manager instance."""
        return self._annotator_service.get_captioner_manager() if self._annotator_service else None

    def get_caption_service(self):
        """Get the caption service instance."""
        return self._annotator_service.get_caption_service() if self._annotator_service else None

    def get_available_captioners(self) -> Dict[str, Dict[str, Any]]:
        """Get information about available caption generators."""
        return self._available_captioners.copy()

    def get_captioner_info(self, captioner_name: str) -> Optional[Dict[str, Any]]:
        """Get information about a specific caption generator."""
        return self._available_captioners.get(captioner_name)

    def list_available_captioners(self) -> List[str]:
        """Get a list of available caption generator names."""
        return list(self._available_captioners.keys())

    def get_ready_captioners(self) -> List[str]:
        """Get a list of ready-to-use caption generator names."""
        return [
            name
            for name, info in self._available_captioners.items()
            if info.get("available", False)
        ]

    def get_info(self) -> Dict[str, Any]:
        """Get service information."""
        info = super().get_info()
        info.update(
            {
                "available_captioners_count": len(self._available_captioners),
                "ready_captioners_count": len(self.get_ready_captioners()),
                "available_captioners": list(self._available_captioners.keys()),
                "ready_captioners": self.get_ready_captioners(),
                "library": "annotator",
            }
        )
        return info

Update Caption Generation Module

Replace yipyap/app/caption_generation/__init__.py with:

"""
Caption Generation Module using Annotator Library

This module provides backward compatibility for existing code while using
the unified annotator library under the hood.
"""

import logging
from typing import Dict, Any, Optional

from annotator import Annotator, CaptionService, get_caption_service

logger = logging.getLogger(__name__)

# Global instances
_annotator: Optional[Annotator] = None
_caption_service: Optional[CaptionService] = None


def get_annotator() -> Annotator:
    """Get the global annotator instance."""
    global _annotator
    if _annotator is None:
        _annotator = Annotator()
    return _annotator


def get_caption_service_instance() -> CaptionService:
    """Get the global caption service instance."""
    global _caption_service
    if _caption_service is None:
        _caption_service = get_caption_service()
    return _caption_service


class CaptionerManager:
    """
    Backward compatibility wrapper for CaptionerManager.
    
    This class provides the same interface as the original CaptionerManager
    but uses the annotator library internally.
    """
    
    def __init__(self):
        """Initialize the captioner manager."""
        self._annotator = get_annotator()
        self._caption_service = get_caption_service_instance()
    
    def get_available_captioners(self) -> Dict[str, Dict[str, Any]]:
        """Get available captioners."""
        return self._annotator.get_available_models()
    
    def get_captioner(self, name: str):
        """Get a specific captioner."""
        try:
            return self._annotator._load_caption_model(name)
        except Exception as e:
            logger.error(f"Failed to load captioner {name}: {e}")
            return None


# Global instance for backward compatibility
captioner_manager = CaptionerManager()


# Export for backward compatibility
__all__ = [
    "CaptionerManager",
    "captioner_manager",
    "get_annotator",
    "get_caption_service_instance",
]

3. Update Pawprint Integration

For pawprint, you can directly use the annotator library:

# In pawprint code
from annotator import Annotator, CaptionService

# Initialize annotator
annotator = Annotator()

# Generate captions
caption = await annotator.generate_caption("path/to/image.jpg", "florence2")

# Or use the service layer
service = CaptionService()
result = await service.generate_single_caption(
    "path/to/image.jpg", 
    "florence2"
)

4. Update Requirements

Add the annotator library to your project's requirements:

# requirements.txt
annotator @ file:///path/to/libraries/annotator

# Or if published to PyPI
annotator>=0.1.0

5. Update Imports

Replace existing imports with annotator library imports:

# Old imports
from app.caption_generation import CaptionerManager
from app.caption_generation.caption_service import CaptionService

# New imports
from annotator import Annotator, CaptionService, CaptionerManager

Testing the Migration

1. Test Basic Functionality

from annotator import Annotator

# Test basic functionality
annotator = Annotator()

# Check available models
models = annotator.get_available_models()
print(f"Available models: {list(models.keys())}")

# Test caption generation (if models are available)
try:
    caption = await annotator.generate_caption("test.jpg", "jtp2")
    print(f"Generated caption: {caption}")
except Exception as e:
    print(f"Error: {e}")

2. Test Service Integration

from annotator import CaptionService

# Test service functionality
service = CaptionService()

# Check available generators
generators = service.get_available_generators()
print(f"Available generators: {list(generators.keys())}")

# Test single caption generation
result = await service.generate_single_caption(
    "test.jpg", 
    "jtp2"
)
print(f"Result: {result}")

Rollback Plan

If issues arise during migration:

  1. Keep old code: Don't delete the original caption generation code immediately
  2. Feature flags: Use environment variables to switch between old and new systems
  3. Gradual rollout: Migrate one component at a time
  4. Monitoring: Add logging to track usage and errors

Benefits of Migration

  1. Unified API: Single interface for all captioning and tagging models
  2. Better error handling: Comprehensive exception system
  3. Improved performance: Smart model loading and caching
  4. Easier maintenance: Centralized codebase
  5. Better testing: Comprehensive test suite
  6. Type safety: Full type hints throughout

Troubleshooting

Common Issues

  1. Import errors: Ensure annotator library is properly installed
  2. Model loading failures: Check model paths and dependencies
  3. Performance issues: Verify model caching is working
  4. Service initialization failures: Check service dependencies

Debug Mode

Enable debug logging:

import logging
logging.getLogger("annotator").setLevel(logging.DEBUG)

Support

For issues with the migration:

  1. Check the annotator library documentation
  2. Review the test suite for usage examples
  3. Check the library's issue tracker
  4. Contact the development team

Oracle - Performance Monitoring and Analysis Core

Oracle is a comprehensive Python library for performance monitoring, analysis, and optimization. It provides real-time metrics collection, health monitoring, performance tuning, and alerting capabilities.

üéØ Features

  • Performance Monitoring: Real-time system and application performance tracking
  • Memory Pressure Detection: Advanced memory monitoring and pressure detection
  • Health Monitoring: Comprehensive dependency and service health checks
  • Performance Tuning: Automated performance optimization suggestions
  • Alerting System: Configurable alerts and notifications
  • Metrics Collection: Flexible metrics collection and aggregation
  • Export Monitoring: Specialized monitoring for data export operations
  • Dashboard Support: Health monitoring dashboard generation

üöÄ Quick Start

Installation

pip install oracle

Basic Usage

from oracle import PerformanceMonitor, RealTimeMonitor
import asyncio

async def main():
    # Create a performance monitor
    monitor = PerformanceMonitor()
    
    # Start real-time monitoring
    realtime = RealTimeMonitor(monitor)
    await realtime.start()
    
    # Monitor your application
    with monitor.track_operation("my_operation"):
        # Your code here
        await asyncio.sleep(1)
    
    # Get performance insights
    insights = await monitor.get_insights()
    print(insights)

asyncio.run(main())

üì¶ Core Components

PerformanceMonitor

The main performance monitoring class that tracks system metrics, application performance, and provides insights.

from oracle import PerformanceMonitor

monitor = PerformanceMonitor(
    collect_system_metrics=True,
    collect_application_metrics=True,
    alert_thresholds={
        "cpu_usage": 80.0,
        "memory_usage": 85.0,
        "disk_usage": 90.0
    }
)

# Track operations
with monitor.track_operation("database_query"):
    # Your database operation
    pass

# Get performance data
metrics = monitor.get_metrics()
insights = monitor.get_insights()

RealTimeMonitor

Provides real-time monitoring capabilities with configurable intervals and alerting.

from oracle import RealTimeMonitor

realtime = RealTimeMonitor(
    monitor=monitor,
    interval=5.0,  # 5 seconds
    enable_alerts=True
)

# Start monitoring
await realtime.start()

# Stop monitoring
await realtime.stop()

MemoryPressureDetector

Advanced memory monitoring with pressure detection and optimization suggestions.

from oracle import MemoryPressureDetector

detector = MemoryPressureDetector(
    pressure_threshold=0.8,
    enable_gc_monitoring=True
)

# Check memory pressure
pressure = detector.check_pressure()
if pressure.is_high:
    suggestions = detector.get_optimization_suggestions()
    print(f"Memory pressure detected: {suggestions}")

PerformanceTuningEngine

Automated performance optimization with intelligent suggestions.

from oracle import PerformanceTuningEngine

tuner = PerformanceTuningEngine(
    monitor=monitor,
    enable_auto_tuning=True
)

# Get optimization suggestions
suggestions = await tuner.analyze_and_suggest()
for suggestion in suggestions:
    print(f"Optimization: {suggestion.description}")
    print(f"Impact: {suggestion.impact}")
    print(f"Difficulty: {suggestion.difficulty}")

DependencyHealthMonitoring

Comprehensive health monitoring for dependencies and services.

from oracle import DependencyHealthMonitoring

health_monitor = DependencyHealthMonitoring(
    services=[
        {"name": "database", "url": "http://localhost:5432/health"},
        {"name": "redis", "url": "http://localhost:6379/health"},
        {"name": "api", "url": "http://localhost:8000/health"}
    ]
)

# Check all dependencies
health_status = await health_monitor.check_all()
for service, status in health_status.items():
    print(f"{service}: {status.status}")

ExportPerformanceMonitor

Specialized monitoring for data export operations.

from oracle import ExportPerformanceMonitor

export_monitor = ExportPerformanceMonitor(
    monitor=monitor,
    track_throughput=True,
    track_errors=True
)

# Monitor export operation
with export_monitor.track_export("data_export", total_items=1000):
    for i in range(1000):
        # Export item
        export_monitor.record_item_exported()
        if i % 100 == 0:
            export_monitor.record_progress(i)

üîß Configuration

Oracle can be configured through environment variables or configuration files:

from oracle import OracleConfig

config = OracleConfig(
    # System monitoring
    system_metrics_interval=5.0,
    enable_cpu_monitoring=True,
    enable_memory_monitoring=True,
    enable_disk_monitoring=True,
    
    # Application monitoring
    application_metrics_interval=1.0,
    enable_operation_tracking=True,
    enable_error_tracking=True,
    
    # Alerting
    alert_enabled=True,
    alert_channels=["console", "email", "webhook"],
    
    # Performance tuning
    auto_tuning_enabled=True,
    tuning_interval=300.0,
    
    # Health monitoring
    health_check_interval=30.0,
    health_check_timeout=10.0
)

üö® Alerting

Oracle supports multiple alerting channels:

from oracle import AlertManager

alert_manager = AlertManager(
    channels=[
        ConsoleAlertChannel(),
        EmailAlertChannel(
            smtp_server="smtp.gmail.com",
            smtp_port=587,
            username="your-email@gmail.com",
            password="your-password"
        ),
        WebhookAlertChannel(
            url="https://your-webhook-url.com/alerts"
        )
    ]
)

# Configure alerts
alert_manager.add_alert(
    name="high_cpu_usage",
    condition=lambda metrics: metrics.cpu_usage > 80,
    message="CPU usage is above 80%",
    severity="warning"
)

üìà Dashboards

Generate health monitoring dashboards:

from oracle import DashboardGenerator

generator = DashboardGenerator(

dashboard.save("performance_dashboard.html")

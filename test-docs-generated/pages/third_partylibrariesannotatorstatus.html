🔧 Migration Process

For Yipyap

  1. Run migration script:

    cd libraries/annotator
python migrate_yipyap.py /path/to/yipyap

  2. Install annotator library:

    cd /path/to/yipyap
pip install -e /path/to/libraries/annotator

  3. Test the integration:

    # Test basic functionality
python -c "from annotator import Annotator; print('Success!')"

For Pawprint

  1. Add annotator to requirements:

    annotator @ file:///path/to/libraries/annotator

  2. Update imports:

    from annotator import Annotator, CaptionService

  3. Test integration:

    annotator = Annotator()
caption = await annotator.generate_caption("image.jpg", "florence2")

Proofreader

A comprehensive validation framework for text, data, and content quality assessment.

Features

  • Unified Validation Framework: Consistent API across all validators
  • Text Validation: Caption and text quality assessment
  • User Input Validation: Password, email, and username validation
  • Image Validation: Quality and format assessment
  • Data Validation: Quality and structure validation
  • Schema Validation: JSON Schema support
  • Extensible: Easy to create custom validators
  • Rich Reporting: Multiple output formats (text, JSON, HTML, Markdown)

Quick Start

from proofreader import CaptionValidator, PasswordValidator, format_validation_report

# Validate a caption
caption_validator = CaptionValidator()
result = caption_validator.validate("A beautiful sunset over the mountains.")
print(format_validation_report(result))

# Validate a password
password_validator = PasswordValidator()
result = password_validator.validate("MyPassword123!")
print(format_validation_report(result))

Alchemist - Media Processing Utilities

A comprehensive Python library for transforming and enhancing images, audio, video, and text through various "alchemical" processes.

🎯 Overview

Alchemist provides a unified interface for media processing tasks, making it easy to convert, enhance, and manipulate different types of media files. The library is designed to be modular, extensible, and easy to use.

✨ Features

Image Processing

  • Format Conversion: Convert between image formats (WebP, AVIF, PNG, JPEG, etc.)
  • Border Removal: Automatically detect and remove white borders from images
  • SVG to PNG: Convert SVG files to PNG with configurable dimensions and backgrounds
  • Grid Plotting: Create comparison grids and before/after plots
  • Sketch Cleaning: Remove white backgrounds and convert grayscale to black

Audio Processing

  • Batch Generation: Process multiple audio files with parallel processing
  • Seed Management: Generate and manage unique seeds for reproducible results
  • Directory Management: Automatic directory creation with proper permissions
  • File Validation: Validate audio files and extract metadata

Video Processing

  • Keyframe Extraction: Extract I-frames from video files
  • Frame Extraction: Extract frames at specific timestamps
  • PNG to MP4: Convert PNG sequences to MP4 videos with effects
  • Video Analysis: Get video metadata and validate video files

Text Processing

  • AI Text Detection: Detect AI-generated text content and identify specific models
  • Parquet Analysis: Analyze and process Parquet data files with summarization
  • Text Analysis: Comprehensive text metrics and statistical analysis
  • Markdown Detection: Identify and process markdown-formatted content

File I/O Utilities

  • Archive Handling: Extract and compress various archive formats (ZIP, TAR, GZ, etc.)
  • Directory Operations: Shuffle, split, clean, and manage directories
  • File Statistics: Get detailed file information, hashes, and comparisons
  • File Validation: Validate files, detect encoding, and find duplicates

🚀 Quick Start

Installation

pip install alchemist

Basic Usage

from alchemist import image, audio, video, text, io

# Convert images to WebP format
from alchemist.image import convert_image
convert_image("input.jpg", ["webp"], quality=85)

# Remove white borders from an image
from alchemist.image import remove_white_border
remove_white_border("sketch.png", threshold=240)

# Extract keyframes from a video
from alchemist.video import extract_keyframes
extract_keyframes("video.mp4", output_format="png")

# Create a video from PNG sequence
from alchemist.video import create_video_from_png_sequence
create_video_from_png_sequence("frames/", framerate=30)

📚 Documentation

Image Processing

Convert Images

from alchemist.image import convert_image, process_directory

# Convert a single image
convert_image("input.jpg", ["webp", "avif"], quality=90)

# Process entire directory
process_directory("images/", ["webp"], quality=80, overwrite=True)

Remove White Borders

from alchemist.image import remove_white_border, has_white_borders

# Remove borders
remove_white_border("image.png", threshold=240)

# Check if image has borders
if has_white_borders("image.png", tolerance=0.1):
    print("Image has significant white borders")

SVG to PNG Conversion

from alchemist.image import svg_to_png, process_directory

# Convert single SVG
svg_to_png("icon.svg", width=512, height=512, bg_color="white")

# Process directory of SVGs
process_directory("icons/", width=256, height=256)

Create Image Grids

from alchemist.image import create_xy_plot, create_comparison_plot

# Create grid plot
create_xy_plot(
    ["img1.jpg", "img2.jpg", "img3.jpg"],
    rows=2,
    column_labels=["Before", "After", "Result"]
)

# Create comparison plot
create_comparison_plot(["img1.jpg", "img2.jpg", "img3.jpg"])

Clean Sketches

from alchemist.image import clean_sketch, get_sketch_analysis

# Clean a sketch
clean_sketch("sketch.png", tolerance=10, gray_tolerance=5)

# Analyze sketch before cleaning
analysis = get_sketch_analysis("sketch.png")
print(f"White pixels: {analysis['white_percentage']:.1f}%")

Audio Processing

Batch Audio Generation

from alchemist.audio import process_audio_generations

def generate_audio(args):
    # Your audio generation logic here
    pass

def prepare_args(description, seeds, output_dir):
    # Prepare arguments for generation
    return [(description, seed, output_dir) for seed in seeds]

# Process multiple descriptions
descriptions = ["A gentle rain", "Ocean waves", "Forest birds"]
process_audio_generations(
    descriptions,
    "my_model",
    generate_audio,
    prepare_args,
    num_variations=10
)

Audio File Management

from alchemist.audio import validate_audio_file, get_audio_file_info

# Validate audio file
if validate_audio_file("audio.wav"):
    info = get_audio_file_info("audio.wav")
    print(f"File size: {info['size_mb']:.2f} MB")

Video Processing

Extract Keyframes

from alchemist.video import extract_keyframes, extract_frames_at_timestamps

# Extract all keyframes
extract_keyframes("video.mp4", output_format="png")

# Extract frames at specific times
timestamps = [0, 5, 10, 15]  # seconds
extract_frames_at_timestamps("video.mp4", timestamps)

Create Videos from Sequences

from alchemist.video import create_video_from_png_sequence

# Create video from PNG sequence
create_video_from_png_sequence(
    "frames/",
    framerate=30,
    scale="1920x1080",
    add_fade=True,
    fade_duration=3
)

Video Analysis

from alchemist.video import get_video_info, validate_video_file

# Get video information
info = get_video_info("video.mp4")
print(f"Duration: {info['duration']} seconds")
print(f"Resolution: {info['width']}x{info['height']}")

# Validate video file
if validate_video_file("video.mp4"):
    print("Valid video file")

Text Processing

AI Text Detection

from alchemist.text import detect_ai_text

# Detect AI-generated text
result = detect_ai_text("sample_text.txt")
print(f"AI probability: {result['ai_probability']:.2f}")

Parquet Analysis

from alchemist.text import analyze_parquet, clean_parquet

# Analyze Parquet file
analysis = analyze_parquet("data.parquet")
print(f"Columns: {analysis['columns']}")

# Clean Parquet data
clean_parquet("input.parquet", "output.parquet")

File I/O Utilities

Archive Operations

from alchemist.io import extract_archive, compress_files

# Extract various archive formats
extract_archive("archive.zip", "output_dir/")
extract_archive("archive.tar.gz", "output_dir/")

# Compress files
compress_files(["file1.txt", "file2.txt"], "archive.zip")

Directory Operations

from alchemist.io import shuffle_directory, split_directory

# Shuffle files in directory
shuffle_directory("data/", seed=42)

# Split directory into parts
split_directory("data/", num_parts=3)

🔧 Configuration

Dependencies

The library requires several external dependencies:

  • Image Processing: Pillow, OpenCV, scikit-image
  • Audio Processing: No external dependencies (uses multiprocessing)
  • Video Processing: ffmpeg (system dependency)
  • Text Processing: pandas, pyarrow
  • File I/O: Standard library only

Optional Dependencies

For enhanced functionality, install optional dependencies:

# For SVG support
pip install cairosvg

# For AVIF support
pip install pillow-avif-plugin

# For development
pip install alchemist[dev]
